# Students App Task

**Project Description:**
A frontend application built with **React**, **TypeScript**, and **Vite** to display and manage student and course data.  
The project uses a **component-based structure** and provides **localStorage** for saving data locally.

**Features:**
- List of students and courses  
- Add, edit, and delete items  
- Local data storage using `localStorage`  
- Simple setup with TypeScript and Vite  

**Requirements:**
- Node.js (v16+ or a version compatible with Vite)
- npm or yarn

**Run Locally (Development):**
```bash
# Install dependencies
npm install

# Run local server (Vite)
npm run dev
